# Kwun Tong 33: Hung To Exorcism

A lightweight 3D browser mini-game set on Hung To Road, Kwun Tong.

## Game Overview
- 5-day survival format (Day 1 to Day 5)
- Each day ends with a boss encounter
- Fixed-position defense gameplay
- Weather and difficulty increase as days progress

## Controls
- Mouse / trackpad: aim
- `S`: fire cross projectile
- `F`: use Holy Water (after unlock)
- `Enter` / `Space` or click `Start`: begin game

## Core Mechanics
- Holy Water unlocks at score 1000
- Every additional 1000 score grants +1 Holy Water charge
- Supply drops can provide HP and utility effects

## Run Locally
1. Open terminal in this folder.
2. Start a local server:

```bash
python3 -m http.server 8080
```

3. Open:

```text
http://localhost:8080
```
